# Personal Website

Simple, minimalist, and responsive personal website.

## Project Structure

```
personal-website
├── index.html         # Main HTML file
├── pgp.html           # HTML file containing my PGP key
├── css

│   └── styles.css      # Custom styles for the website
├── assets
│   └── favicon.ico     # Favicon for the website
└── README.md
```
